# YouTube-Video-Downloader
Developed a desktop application using Python and Tkinter that enables users to download YouTube videos through a graphical interface. Integrated the yt-dlp library for video extraction and implemented file management and error-handling mechanisms.

YouTube Video Downloader using Python

Developed a desktop application using Python and Tkinter to download videos from online platforms.

Integrated yt-dlp for video extraction and downloading.

Implemented a user-friendly GUI for URL input and download management.

Applied exception handling and file management techniques.
